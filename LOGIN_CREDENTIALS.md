## Quick Reference

### To login as Admin:
1. Go to http://127.0.0.1:5000
2. Select "Admin" login type
3. Enter username (e.g., `admin1`) and password (`admin123`)

### To login as Driver:
1. Go to http://127.0.0.1:5000
2. Select "Driver" login type
3. Enter username (e.g., `driver1`) and password (`pass123`)
